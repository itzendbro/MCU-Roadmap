# 🎬 Marvel Master Release Tracker

A clean, responsive web interface to track and access the Marvel Cinematic Universe (MCU) and related Marvel films in chronological release order.

## ✨ Features
*   **Release Tracking:** Stay updated with the latest MCU releases, including upcoming 2025 titles.
*   **Quick Links:** Direct access to media content for movies and series.
*   **Optimized Layout:** Dark mode aesthetic with high-contrast elements for easy reading.
*   **Pinned Classics:** Non-MCU essentials (Tobey Maguire/Andrew Garfield Spider-Man and 2003 Hulk) pinned to the top.

## 💡 Recommendation
For the best experience, it is highly recommended to view this site using **Brave Browser** to ensure an ad-free and streamlined navigation experience.

*   HTML5
*   CSS3 (Custom styling & Responsive design)
